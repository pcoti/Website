I spend kind of a lot of time programming, mostly in support of my research. My normal stack is R, SQL, Hadoop and Hive, with bits of Python, AWK and C++ on occasion. When I'm not doing research I tend to invest my time in building useful and not-so-useful R libraries. These include:

WikipediR

A lot of programming languages have MediaWiki API wrappers, from Python to Ruby, R wasn't one of them, so I wrote WikipediR, which is aimed particularly at the Wikimedia-run instances of MediaWiki.

averageimage

In an attempt to answer the question "what's the stupidest thing it's possible to develop exclusively using a language built for statistical analysis?" I wrote averageimage, a composite image generator in R that accepts JPEGs and both colour and greyscale PNGs, trims them according to the rule of thirds, and generates and outputs a composite. An example can be seen on the left.

WMUtils

As an aid to my work at the WMF, I wrote WMUtils, a library of WMF-centred R functions. It includes Hive querying, Python script integration, and streaming to and from the MaxMind geolocation API. It also integrates with the ua-parser library, which I help maintain in my spare time.
